<semantic-list>

Selectable lists with keyboard navigation and ARIA support

First Item Second Item Selected Item

Overview

The semantic-list component displays items in a vertical list with optional selection. Lists automatically handle keyboard navigation (arrow keys), focus management, and multi-select when enabled.

Examples

Selectable List
First Item Second Item Selected Item
<semantic-list selectable> <list-item>First Item</list-item> <list-item>Second Item</list-item> <list-item aria-selected="true">Selected Item</list-item> </semantic-list>
Use for: Option selection, file lists, settings
Keyboard: Arrow keys navigate, Space/Enter select
Accessibility: ARIA selection states announced
