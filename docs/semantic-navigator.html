<semantic-navigator>

Multi-modal navigation with tabs, breadcrumbs, and spatial layouts

Home Profile Settings

Overview

The semantic-navigator component handles navigation UI patterns with support for tabs, sidebars, breadcrumbs, and spatial (AR/VR) navigation. It automatically highlights the current page.
