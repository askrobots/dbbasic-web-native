<semantic-adjuster>

Multi-modal number/value adjusters with voice, gesture, and button controls

Overview

The semantic-adjuster component provides increment/decrement controls for numeric values. It supports button clicks, voice commands ("increase", "decrease"), gestures, and keyboard arrows.

Examples

Volume Control
<semantic-adjuster label="Volume" value="75" min="0" max="100" step="5"></semantic-adjuster>
Use for: Volume, brightness, quantity adjustments
Voice: "Increase volume", "Decrease volume"
Gestures: Swipe up/down in AR/VR
Temperature Setting
<semantic-adjuster label="Temperature" value="20" min="15" max="30" step="0.5"></semantic-adjuster>
Use for: Numeric settings, sliders, counters
Keyboard: Arrow up/down to adjust
Touch: Tap +/- or swipe
