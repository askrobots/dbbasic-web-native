Overview

The semantic-modal component displays overlay dialogs with attention budget awareness. Modals automatically adjust their urgency presentation and can be dismissed by clicking the backdrop or pressing Escape.

Use for: Confirmation dialogs, forms, alerts
Visual: Dark background, centered, backdrop blur
Interaction: Click backdrop or press Escape to dismiss

Properties & Attributes

Attribute Type Values Default Description
open boolean true | false false Whether the modal is currently visible
urgency string critical | high | medium | low medium Visual urgency level (critical shows red border)
dismissible boolean true | false true Whether modal can be dismissed by backdrop click/Escape
header slot Modal title/header content
default slot Main modal body content
actions slot Modal footer with action buttons

Events

Event Trigger Detail Description
