Overview

The semantic-card component displays content in a structured format with optional media, title, description, preview content, and actions. It automatically handles hover elevation and preview display.

Use for: Content previews, product cards, article teasers
Visual: Dark background, hover elevation
Interaction: Click to activate, long-press to preview

Properties & Attributes

Attribute Type Values Default Description
media-url attribute URL URL for image media
media-alt attribute text Alt text for image
media-icon slot Icon/emoji when no image
title slot Card title
description slot Card description/subtitle
preview slot Content shown on hover/long-press
actions slot Action buttons

Events

Event Trigger Detail Description
activate Click on card { type: "activate" } Card click/tap event
inspect Long-press (500ms) { type: "inspect" } Shows preview content
